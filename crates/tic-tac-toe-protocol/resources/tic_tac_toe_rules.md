# Tic-Tac-Toe Rules

Two players take turns placing marks (X or O) on a 3x3 grid. X always goes first. The first player to get three marks in a row (horizontally, vertically, or diagonally) wins. If all nine cells are filled with no winner, the game is a draw.

Moves use zero-based (row, col) coordinates: (0,0) is the top-left cell, (2,2) is the bottom-right cell.
